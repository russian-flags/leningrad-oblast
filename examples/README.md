# Example

Демо-приложение показывает, как использовать пакет `@russian-flags/leningrad-oblast` в обычном frontend-проекте.

Пример работает как отдельное Vite-приложение: он устанавливает пакет из npm и импортирует его так же, как это будет делать внешний проект.

## Запуск

```bash
cd examples
npm install
npm run dev
```

После запуска Vite откроет страницу в браузере. Если браузер не открылся автоматически, открой адрес из терминала, обычно это:

```text
http://127.0.0.1:5173/
```

Не открывай `index.html` двойным кликом: npm-импорты и Vite-модули работают через dev-сервер.

## Что проверяет пример

- импорт `loadFlag`, `preloadFlag` и `settlements` из npm-пакета;
- ленивую загрузку SVG-флагов;
- отображение списка населенных пунктов;
- открытие флага в модальном окне.

## Сборка

```bash
npm run build
```

Собранные файлы появятся в `examples/dist`.

## Preview сборки

```bash
npm run preview
```

## Запуск из корня репозитория

Из корня проекта можно запустить те же команды через `--prefix`:

```bash
npm --prefix examples install
npm run example:dev
```

Для проверки production-сборки:

```bash
npm run example:build
```
